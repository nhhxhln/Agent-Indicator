# Agent Indicator — 电源方案设计

> Rev 0.2 · English version: [en/02-power-design.md](en/02-power-design.md) · 配套 [01-system-design.md](01-system-design.md)

## 1. 电源拓扑

```
USB-C PD ──► CH224K ──► VPD(15V≥3A) ──► 理想二极管 ─┐
  (诱骗15V)                            (LM5050-1+PFET)│
                                                      ├──► VIN_SEL ──► MP2760 ──► VSYS(9~12.6V)
XT30 ──► 防反接PFET+TVS ──► VIN(12-24V) ─► 理想二极管 ─┘            (NVDC buck-boost   │
                                          (优先级高)                 充电器+电源路径)   │
                                                                        ▲▼              │
                                            3S 18650 ◄── S-8254A 保护 ──┘               │
                                            (11.1-12.6V)  HY2213×3 均衡                 │
                                                                                        │
        ┌───────────────────────────────────────────────────────────────────────────────┤
        │                                     │                                         │
   TPS56637                               MP2315S                                       │
   VLED 5.0V/5A ──► PMOS开关(LED_PWR_EN)  5V0 2A ──┬─► TPS2121 ◄── VUSB(USB-C 数据口VBUS)
   ──► WS2812B Matrix/Ring/Bars                    │   (无缝切换+防倒灌,USB可单独点亮逻辑)
                                                   ├─► SGM2212-3.3 ──► VDD 3.3V
                                                   │    (SD卡/IMU/Codec逻辑/扩展I2C)
                                                   └─► 磁珠+钽电容 ──► VAUDIO 5V(NS4150B)
```

要点:
- **MP2760 NVDC 电源路径**:输入在位时系统由输入供电并给电池充电,输入移除瞬时无缝切到电池,无需额外切换电路。
- **输入优先级**:XT30 与 VPD 经理想二极管 ORing。XT30 支路加 30mV 偏置使同压时 XT30 优先;PD 仅 15V、XT30 24V 时自然高压优先。
- **VLED 与 5V0 隔离**:两路独立 buck,LED 大电流纹波不串入逻辑电源;USB 数据口 VBUS 经 TPS2121 与 5V0 互为备份且互不倒灌(拔掉主电源时插 USB 仍可调试,LED 不上电)。

## 2. 器件选型

| 节点 | 器件 | 关键参数 | 说明 |
|---|---|---|---|
| PD 诱骗 | **CH224K** | 申请 15V(CFG 配置),100W 级 | 15V > 12.6V+充电裕量,满足 buck-boost 全范围;LED 脚指示 PD 成功 |
| 输入 ORing | **LM5050-1** ×2 + PFET(如 AO4407A) | 理想二极管控制器 | 也可用集成款(如 LM74700) |
| XT30 防反/浪涌 | PFET 防反接 + **SMBJ26A** TVS + NTC | 24V 输入 | XT30 旁出 CAN 4P 端子(CANH/L/12V/GND) |
| 充电器 | **MP2760** | 1-4S NVDC buck-boost,I2C,输入 2.7-22V,充电 ≤6A | 12V XT30 / 15V PD 都能给 3S 充满;I2C 上报状态;备选 BQ25756(控制器+外置管,功率更大) |
| 电池保护 | **S-8254A** + 双 NMOS(共漏) | 3S 过充/过放/过流/短路 | 经典方案;备选 BQ76905(带库仑计) |
| 均衡 | **HY2213-BB3A** ×3 | 单节 4.2V 旁路均衡 66mA | 每节并联,简单可靠的充电末端均衡 |
| 电池监测 | 3 节分压 → MCU ADC + **INA226**(VSYS 高侧) | — | SOC 估算 + 功耗遥测 |
| VLED buck | **TPS56637** | 4.5-28V 输入,6A,固定 5V 可调 | 输出 5.0V,布局靠近 LED 接口;前级 PMOS 受 LED_PWR_EN 控制 |
| 5V0 buck | **MP2315S** | 4.5-24V,3A | 给开发板/逻辑/音频 |
| USB 防倒灌 | **TPS2121** | 双输入功率 mux,无缝切换 | 5V0(优先)vs VUSB;输出 5V0_SYS |
| VDD LDO | **SGM2212-3.3** | 1A,低压差 | SD 卡峰值 ~200mA + 外设充裕 |
| VAUDIO | 磁珠 + 220µF | 自 5V0_SYS 滤波 | NS4150B 3W 突发电流去耦 |

## 3. 功耗预算

WS2812B 按实测典型 16mA/通道(5V 折算 240mW/颗全白)计;固件全局限流到 VLED 4.5A。

| 负载 | 数量 | 典型(动画 25% 亮度) | 峰值(全白) |
|---|---|---|---|
| Matrix 8×8 | 64 | 0.77A | 3.07A |
| Matrix 16×16(扩展) | 256 | 3.07A | 12.3A → 固件限流 |
| Circle | 24 | 0.29A | 1.15A |
| Usage bar 20 + 拾音条 64 | 84 | 1.0A | 4.0A |
| **VLED 合计(8×8 配置)** | 172 | **~2.1A / 10.4W** | 限流 4.5A / 22.5W |
| LCD(屏+背光) | — | 1.2W | 1.5W |
| ESP32-S3(Wi-Fi 活动) | — | 0.8W | 1.7W(峰值) |
| 音频(3W PA 突发) | — | 0.2W | 3W |
| **系统合计** | — | **~14W** | ~29W |

- PD 15V×3A=45W:供整机峰值 + 约 1A 充电电流,够用;XT30 24V 时充电可放开到 2A。
- 续航估算:3S×3000mAh≈33Wh,典型 14W → **~2.3h**;待机模式(LED 断电、LCD 低亮)<1.5W → >20h。
- 16×16 扩展:建议外接 XT30 供电使用,VLED 限流策略将全白场景压到 4.5A(平均亮度 ~35%),如需更高可并联第二片 TPS56637(预留焊位)。

## 4. 保护与细节

1. **过充/过放**:S-8254A 阈值 4.25V/2.7V 每节;MP2760 充电终止 12.6V、截止电流 C/10。
2. **平衡充**:HY2213 在任一节到 4.2V 时旁路 66mA,配合 MP2760 末端小电流完成均衡。
3. **防倒灌**:USB VBUS 仅进 TPS2121,永不反灌;VPD/VIN 理想二极管单向;VLED 关断 PMOS 防 LED 灯板反向漏电。
4. **浪涌/ESD**:XT30 进口 TVS;USB-C 两口 USBLC6-2SC6;CAN 口 PESD1CAN。
5. **热设计**:TPS56637 大面积铺铜 + 过孔阵;MP2760 充电 2A 时 ~1W 损耗,同样处理。
6. **上电时序**:VDD 先于 LED_PWR_EN/BL_EN(由固件经 TCA9554 延时使能),避免 WS2812 上电窜亮。
7. **采样**:3 节电池分压经 100k/10k + RC 进 ADC(经保护板前的均衡抽头),静态泄放 <15µA。

| 接口 | 连接器 | 引脚 |

| PD 输入 | USB-C 16P(仅 PD) | CC 接 CH224K |
| 数据 USB | USB-C 16P | D+/D- → GPIO19/20,VBUS → TPS2121 |
| 外部电源 | XT30-M | 12-24V |
