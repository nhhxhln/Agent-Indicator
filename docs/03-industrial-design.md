# Agent Indicator — 外观与结构设计(4 方案)

> English version: [en/03-industrial-design.md](en/03-industrial-design.md)
> 3D 模型源文件与可打印 STL:[`hardware/3d/`](../hardware/3d/)(OpenSCAD)

> Rev 0.1。核心设计语言:**Halo 单元** —— LCD(AA 70.13mm 方形)沉于 LED Circle(ID75/OD85)正后方,
> 环体遮住屏幕四角(AA 对角线 99.2mm > 75mm 孔径),配合圆形 UI,视觉上呈现一块"会发光呼吸的圆屏"。
> 环到屏面留 3~4mm 段差 + 内壁哑光黑,环光不漏进屏面。

通用工艺:外壳 CNC 阳极铝或树脂 3D 打印喷砂;灯面 2mm 奶白亚克力(透光率 ~40%)做扩散,
灯珠到扩散面 ≥8mm 并加格栅(matrix 每格 8×8mm 井字格,杜绝串光);底部硅胶垫 + 配重。

## 方案 A 「Halo」— 圆形单体,最紧凑

只含 Halo 单元 + 底座,Usage 画在 LCD 圆形 UI 边缘弧段,Context 以圆环热力图呈现(无 Matrix)。
适合做"最小可爱款";Matrix 可后期以方案 D 模块磁吸外挂。

## 方案 C 「Totem」— 竖向图腾,占桌面最小

纵向叠放,正投影仅 78×62mm。从上到下:Halo → Matrix → 底座;Usage 竖条贴右侧棱,拾音条嵌底座前缘。

```
      ┌──────────┐
      │ ╭──────╮ │
      │ │ Halo │ │  Ø85 → 段宽 90
      │ ╰──────╯ │
      ├──────────┤▌
      │ ▦▦▦▦▦▦▦▦ │▌← Usage 20LED
      │ ▦ 8×8  ▦ │▌   贴右棱竖排
      │ ▦▦▦▦▦▦▦▦ │▌
      ├──────────┤
      │ ▂▃▅▆▅▃▂  │ ← 底座:电池+主板+SPK
      └──────────┘    前缘拾音条 20LED@41.5mm
       78 × 240(高) × 62mm,底座配重 150g
```

| 项 | 值 |
|---|---|
| 整机 | 78 × 240 × 62mm(底座深 90mm 防倾倒) |
| 拾音条 | 20LED @ 41.5mm(底座宽度限制) |
| 电池 | 3×18650 竖置底座 |
| 特点 | 桌面占地最小;Halo 在视线高度;IMU 检测倾倒自动断 VLED |

## 方案 D 「Tiles」— 磁吸模块系统,可玩性最高

统一 **90×90×22mm** 模块外框(容纳最大件:Ring OD85 / LCD 76.48),四边中点磁吸 + 4P pogo
(VLED / GND / DATA-IN / DATA-OUT),底部燕尾槽卡入供电轨。Matrix 模块 2×2 拼即得 16×16。

```
  ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐
  │ ╭────╮ │ │ ▦▦▦▦▦▦ │ │   ▌    │ │ ◌◌◌◌◌◌ │
  │ │Halo│ │ │ ▦ 8×8▦ │ │   ▌Bar │ │ ◌ 空白◌│  ← 模块家族:
  │ ╰────╯ │ │ ▦▦▦▦▦▦ │ │   ▌    │ │ ◌(扩展)│    Halo / Matrix / Bar / 自定义
  └──┬─┬───┘ └──┬─┬───┘ └──┬─┬───┘ └──┬─┬───┘
  ═══╧═╧════════╧═╧════════╧═╧════════╧═╧══════  ← Dock 供电轨 370×60×26mm
  │ ▂▃▅ 拾音条 80LED@164.7mm ▅▃▂   [主机区] │     内置:主板+电池+SPK+MIC
  ╚═══════════════════════════════════════════╝     右端主机区出 USB-C/XT30/CAN
```

| 项 | 值 |
|---|---|
| 模块 | 90 × 90 × 22mm,磁吸力 ~0.8kg(Ø8×3 钕磁 ×4) |
| Dock | 370 × 60 × 26mm(4 模块位),或 190mm 双位短轨 |
| 拾音条 | 80LED @ 164.7mm 嵌 Dock 前缘(短轨用 48LED@99mm) |
| 电气 | 轨内 VLED 总线 + WS2812 数据菊花链经 pogo 串行流过各模块;模块内 EEPROM(24C02)存类型 ID,主机上电枚举排序 |
| 16×16 | 4 个 Matrix 模块 2×2 离轨磁吸拼接,边缘 pogo S 形跳数据,整体再以 1 个轨位接入 |

## 方案对比与建议

| | A Halo | B Console | C Totem | D Tiles |
|---|---|---|---|---|
| 功能完整度 | ◐(无 Matrix) | ● | ● | ●+ |
| 紧凑/精致 | ●● | ● | ●(占地最小) | ◐(轨道偏长) |
| 结构难度 | 低 | 中 | 中 | 高(pogo/磁吸/枚举) |
| 16×16 扩展 | 外挂 | 换壳 | 不支持 | 原生 |
| 建议定位 | 入门款/礼品 | **主力款,先做** | 衍生款 | 旗舰款,二期 |

**建议路线**:先出 B(Console)验证全部电路与固件 → 复用 Halo 单元出 A → 二期做 D 的 Dock 与模块化结构。
四个方案共用同一块主板(见 04 文档),仅外壳与 LED 载板不同。

待确认尺寸(我按常规估的,如有实物请修正):
1. 18650 含保护板长度按 65mm(无保护)算,支架外廓 58×78mm;
2. Matrix 灯板厚度按 PCB 1.6mm + 灯珠 2.5mm 算;
3. LCD 模组总厚按 3.5mm(含 CTP)算。
